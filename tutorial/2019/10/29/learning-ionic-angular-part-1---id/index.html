    Tutorial IONIC dengan Angular Framework - Part #1 (Persiapan)

    Tutorial
    October 29, 2019 Aditya Chandra

    JIKA KAMU INGIN MENGETAHUI RINGKASAN MATERI ARTIKEL INI. KAMU BISA LIHAT BAGIAN KESIMPULAN PADA BAGIAN PALING BAWAH ARTIKEL.

    Pengenalan

    IONIC adalah framework untuk Hybrid app development. IONIC adalah framework yang ringan, jadi ga perlu khawatir dengan komputermu (jika memiliki spesifikasi kecil), setidaknya kamu menggunakan RAM 2GB dan Hardisk 50 GB sudah cukup.

    Yang paling utama adalah software pendukung yang diperlukan untuk proses development IONIC App :

    • Node & NPM - Instal di sini
    • Text Editor - VS Code
    • Pengetahuan dasar tentang menjalankan perintah cmd / terminal
    • Pengetahuan tentang javascript
    • Pengetahuan tentang framework javascript, yaitu Angular.

    Pada artikel ini dan selanjutnya, saya akan menggunakan VS Code dan menggunakan framework Angular.

    1. Instalasi NPM

    Kamu bisa membaca Dokumentasi NPM di sini untuk melakukan proses instalasi. Untuk memastikan bahwa instalasi berhasil, kamu bisa menjalankan : npm -v dan node --version.

    Ini contoh output dari terminal :

    $ npm -v
6.12.0

$ node --version
v10.16.3

    Jika 2 perintah di atas berhasil menampilkan versi npm dan node tanpa error. Kamu bisa lanjut ke langkah berikutnya.

    2. Instalasi IONIC

    Kamu bisa membaca Dokumentasi IONIC di sini untuk melakukan proses instalasi IONIC CLI.

    Singkatnya, jika npm sudah terpasang di komputermu, maka kamu hanya perlu menjalankan perintah : npm install -g ionic. Kemudian, untuk mengetahui apakah ionic telah terpasang, kamu bisa menjalankan perintah : ionic -v di terminal.

    $ ionic -v
5.4.1

    3. Instalasi Angular CLI

    Proses selanjutnya adalah instalasi Angular CLI bisa dilihat di sini. Kamu hanya perlu menjalankan perintah npm install -g @angular/cli.

    Pastikan instalasi Angular CLI berhasil dengan menjalankan perintah ng version.

    Akan keluar output seperti di bawah ini :

    4. Plugin Visual Studio Code

    Pada artikel ini dan selanjutnya, saya menggunakan Visual Studio Code sebagai teks editor. Kamu bisa instal plugin Angular Essentials untuk memudahkan proses development aplikasi ionic, lihat di sini.

    Untuk mempelajari cara memasang ekstensi di Visual Studio Code, kamu bisa membaca dokumentasi resmi Visual Studio Code di link berikut.

    Kesimpulan

    Artikel ini membahas tentang persiapan apa saja sebelum kita memulai untuk ngoding ionic. Yaitu, instalasi beberapa program berikut :

    1. Instalasi Node & NPM - NodeJs.org
    2. Instalasi Ionic Framework - Ionic CLI Docs
    3. Angular CLI - Angular CLI Docs
    4. Memasang plugin Angular Essentials di Visual Studio Code - Cara memasang ekstensi, Angular Essentials

    Setelah proses intalasi program - program di atas selesai. Kita bisa mulai mencoba untuk mempelajari Angular dan Ionic.
