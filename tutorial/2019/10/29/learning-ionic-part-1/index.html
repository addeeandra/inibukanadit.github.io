    Learning IONIC Framework - Part #1

    Tutorial
    October 29, 2019 Aditya Chandra

    Introduction

    IONIC is a mobile application development framework for Hybrid Apps using Javascript based on Cordova. Within this series I'll use ionic-angular codebase. And our first steps are the installation and preparation process for further development.

    Preparation

    IONIC is a lightweight frameworks, so don't worry with your computer, at least you have 2GB RAM and 50 GB Disk Drive are enough, though. The software requirements are more important, here are the list :

    • Node & NPM - Install here
    • Text Editor (You may use VS Code, Sublime or Notepad++)
    • Basic knowledge for CMD / Terminal commands
    • Fluent knowledge with Javascript (or at least, understands the Object Oriented Programming)

    NPM Installation

    You could read the NPM docs here for the installation process. Just ensure that the installation are correct by runing these commands : npm -v and node --version.

    If those 2 commands return the expected outputs without any error. You'll be ready to go!
